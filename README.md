# TP-Integrador
# Descripción

# En este proyecto la aplicación del programa permite gestionar información de distintos países mediante la lectura de un archivo CSV. El sistema implementa funciones de búsqueda, filtrado, ordenamiento y generación de estadísticas utilizando estructuras de datos de Python.

# Funcionalidades
# - Carga de datos desde archivo CSV.
# - Agregar nuevos países.
# - Actualizar población y superficie de un país.
# - Buscar países por nombre.
# - Filtrar países por:
#     - Continente.
#     - Rango de población.
#     - Rango de superficie.
# - Ordenar países por:
#     - Nombre.
#     - Población.
#     - Superficie.
# - Mostrar estadísticas:
#     - País con mayor población.
#     - País con menor población.
#     - Promedio de población.
#     - Promedio de superficie.
#     - Cantidad de países por continente.
